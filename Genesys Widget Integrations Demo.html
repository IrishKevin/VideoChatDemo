Genesys Widget Integrations Demo

VideoEngager Integrations

When you load genesys widget, it is mandatory to refresh the page before trying another configuration.

SmartVideo SDK Parameters

PureCloud agent will receive interaction when is "On Queue" with queue from Queue Name
To get your Organization Id open PureCloud as admin and navigate to Admin > Account Settings > Organization Settings > Organization Details (tab) > Advanced > Organization ID
To get your Deployment Id open PureCloud as admin and navigate to Admin > Contact Center > Widgets > Deployment Key (column)
To get your Tenant ID select Apps > SmartVideo_Settings > Tenant ID
VideoEngager Server Url
Select PureCloud Region

You can save current Advanced Settings to Local Storage. They can be reused from the same browser from the same computer many times. Clear will remove settings from Local Storage and Input Fields

Please use your Genesys Cloud and VideoEngager configuration parameters here.

To try Schedule Video, please configure schedule meeting in your Genesys Organisation.
